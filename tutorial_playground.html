🎯 Цель работы: Что, на чем и для чего мы обучаем?

Перед тем как приступить к написанию кода, давайте разберем саму суть нашего эксперимента на понятном примере:

📊
1. НА ЧЕМ мы обучаемся? (Входные данные)
Мы работаем с цифровой картой, на которой разбросаны точки двух цветов:
  • Красные точки (Класс 0): опасные зоны (например, «очаги лесных пожаров»).
  • Зеленые точки (Класс 1): безопасные зоны (например, «зеленые поля»).
У каждой точки на нашей карте есть ровно два признака — её координаты: X1 (широта, положение по горизонтали) и X2 (долгота, положение по вертикали).
🧠
2. ЧТО мы обучаем? (Процесс)
Мы создаем и обучаем нейронную сеть. В самом начале она «слепа» и не знает, в каких координатах находятся пожары, а в каких — безопасность.

В процессе обучения (по эпохам) нейросеть анализирует координаты точек и их цвета, пытаясь провести математическую разделительную границу между красными и зелеными областями. Поскольку точки расположены нелинейно (в форме полумесяцев или кругов), сеть учится изгибать эту границу, формируя сложную карту разделения.
🏆
3. ДЛЯ ЧЕГО мы это делаем? (Итог и польза)
Наша цель — получить обученную модель ИИ, способную принимать самостоятельные решения в будущем.

Когда обучение завершено, мы можем ввести координаты любой новой точки (которой изначально не было в нашей базе). Нейросеть мгновенно проанализирует координаты и выдаст вердикт: безопасна ли эта область или там бушует пожар, а также укажет точный процент своей уверенности в ответе.

  • streamlit — превращает наш скрипт Python в полноценный веб-сайт с ползунками, вкладками и графиками прямо в браузере.
  • pandas — библиотека для анализа данных. Хранит координаты точек в удобных таблицах (DataFrame).
  • numpy — библиотека быстрой математики и многомерных массивов (из которых строятся нейросети).

  • scikit-learn — генерирует геометрические фигуры точек и рассчитывает аналитические метрики качества.
  • tensorflow — флагманская библиотека для построения сложных нейросетей от Google.

📊 Шаг 2 — Создание модуля генерации данных (data_manager.py)

Этот файл отвечает за создание искусственных точек и деление их на обучающую и проверочную выборки. Создайте в PyCharm файл data_manager.py (правой кнопкой на папку проекта -> New -> Python File) и добавьте этот код:

data_manager.py python 
import numpy as np
import pandas as pd
from sklearn.datasets import make_moons, make_circles, make_blobs
from sklearn.model_selection import train_test_split

class DataManager:
    """
    Класс для генерации, предварительной обработки и разделения наборов данных.
    """
    @staticmethod
    def generate_dataset(dataset_type: str, n_samples: int, noise: float):
        # Генерация двумерных наборов данных для классификации
        if "Полумесяцы" in dataset_type:
            X, y = make_moons(n_samples=n_samples, noise=noise, random_state=42)
        elif "Круги" in dataset_type:
            X, y = make_circles(n_samples=n_samples, noise=noise, factor=0.5, random_state=42)
        else: # Облака / Блоки (Мультиклассовая классификация)
            X, y = make_blobs(n_samples=n_samples, centers=2, cluster_std=noise * 5, random_state=42)
        return X, y

    @staticmethod
    def prepare_data(X: np.ndarray, y: np.ndarray, test_size: float):
        # Разделяет данные на обучающую и тестовую выборки
        X_train, X_test, y_train, y_test = train_test_split(
            X, y, test_size=test_size, random_state=42
        )
        return X_train, X_test, y_train, y_test

    @staticmethod
    get_metrics_summary(df: pd.DataFrame):
        # Расчет распределения классов в Pandas DataFrame
        summary = df["target"].value_counts().to_dict()
        return summary

📝 Подробный разбор кода data_manager.py

  • Импорт модулей: Импортируем `numpy` для математических массивов и `pandas` для анализа таблиц. Из библиотеки `scikit-learn` берутся генераторы нелинейных фигур (`make_moons` и `make_circles`) и разделитель данных.
  • Статические методы (@staticmethod): Декоратор позволяет вызывать функции класса напрямую (`DataManager.prepare_data(...)`) без необходимости создавать экземпляр класса через `manager = DataManager()`.
  • Входные аргументы generate_dataset: Принимает тип фигуры (`dataset_type`), количество точек (`n_samples`) и уровень зашумленности (`noise`). Функция возвращает кортеж из матрицы координат `X` (размером N строк на 2 столбца) и вектора правильных ответов `y` (содержит 0 или 1 для каждой точки).
  • Разбиение Train/Test: Функция `train_test_split` делит датасет на две части: обучающую (на которой сеть учится подгонять веса) и тестовую (скрытую, по которой оценивается реальная точность ИИ). Параметр `test_size` задает долю тестовой выборки (например, `0.2` означает 20% данных). `random_state=42` фиксирует случайное перемешивание, чтобы при каждом запуске программы точки генерировались одинаково.

🧠 Шаг 3 — Создание математического ядра классификации (model_engine.py)

Этот файл содержит всю математическую логику: создание нейросети Keras/Scikit-learn, цикл пошагового обучения по эпохам на CPU, расчет границ разделения плоскости и выгрузку весов. Создайте в PyCharm Python-файл model_engine.py и добавьте следующий код:

model_engine.py python 
import numpy as np
import joblib
from sklearn.neural_network import MLPClassifier
from sklearn.metrics import accuracy_score, precision_score, recall_score, confusion_matrix

# Безопасный импорт TensorFlow для работы на любом компьютере
try:
    import tensorflow as tf
    from tensorflow.keras.models import Sequential
    from tensorflow.keras.layers import Dense, Dropout
    from tensorflow.keras.optimizers import Adam
    keras_available = True
except ImportError:
    keras_available = False

class ModelEngine:
    def __init__(self, engine_type: str, hidden_layers: tuple, activation: str, learning_rate: float):
        self.engine_type = engine_type if (engine_type != "TensorFlow/Keras" or keras_available) else "Scikit-learn (MLP)"
        self.hidden_layers = hidden_layers
        self.activation = activation
        self.learning_rate = learning_rate
        self.model = None
        self.is_trained = False
        
        self.keras_activation_map = {
            "logistic": "sigmoid", "relu": "relu", "tanh": "tanh", "identity": "linear"
        }

    def initialize_model(self, input_dim: int):
        if self.engine_type == "TensorFlow/Keras" and keras_available:
            self.model = Sequential()
            # Создание Keras модели с динамическим числом слоев на основе ползунков
            self.model.add(Dense(self.hidden_layers[0], activation=self.keras_activation_map.get(self.activation, "relu"), input_shape=(input_dim,)))
            for neurons in self.hidden_layers[1:]:
                if neurons > 0:
                    self.model.add(Dense(neurons, activation=self.keras_activation_map.get(self.activation, "relu")))
            self.model.add(Dense(1, activation="sigmoid"))
            self.model.compile(optimizer=Adam(learning_rate=self.learning_rate), loss="binary_crossentropy", metrics=["accuracy"])
        else:
            # Создание Scikit-learn MLP модели
            self.model = MLPClassifier(
                hidden_layer_sizes=self.hidden_layers, activation=self.activation,
                learning_rate_init=self.learning_rate, max_iter=1, random_state=42, warm_start=True
            )

    def train_epoch(self, X_train: np.ndarray, y_train: np.ndarray, classes: np.ndarray = None):
        if self.engine_type == "TensorFlow/Keras" and keras_available:
            history = self.model.fit(X_train, y_train, epochs=1, batch_size=4, verbose=0)
            loss = history.history['loss'][0]
            acc = history.history['accuracy'][0]
        else:
            # Обучение MLP на одну эпоху с помощью partial_fit
            self.model.partial_fit(X_train, y_train, classes=classes)
            loss = self.model.loss_
            acc = self.model.score(X_train, y_train)
        return loss, acc

    def evaluate(self, X_test: np.ndarray, y_test: np.ndarray):
        if self.engine_type == "TensorFlow/Keras" and keras_available:
            preds_prob = self.model.predict(X_test, verbose=0)
            preds = np.where(preds_prob >= 0.5, 1, 0).flatten()
        else:
            preds = self.model.predict(X_test)
            
        accuracy = accuracy_score(y_test, preds)
        precision = precision_score(y_test, preds, zero_division=0)
        recall = recall_score(y_test, preds, zero_division=0)
        conf_matrix = confusion_matrix(y_test, preds)
        
        self.is_trained = True
        return {
            "accuracy": accuracy, "precision": precision, "recall": recall, "confusion_matrix": conf_matrix
        }

    def get_decision_boundary(self, X: np.ndarray, resolution: int = 50):
        # Генерация координатной сетки для отрисовки тепловой карты границ ИИ
        x_min, x_max = X[:, 0].min() - 0.5, X[:, 0].max() + 0.5
        y_min, y_max = X[:, 1].min() - 0.5, X[:, 1].max() + 0.5
        xx, yy = np.meshgrid(np.linspace(x_min, x_max, resolution), np.linspace(y_min, y_max, resolution))
        grid_points = np.c_[xx.ravel(), yy.ravel()]
        
        if self.engine_type == "TensorFlow/Keras" and keras_available:
            Z = self.model.predict(grid_points, verbose=0).flatten()
        else:
            Z = self.model.predict_proba(grid_points)[:, 1]
        return xx, yy, Z.reshape(xx.shape)

    def predict_point(self, point: np.ndarray):
        if self.engine_type == "TensorFlow/Keras" and keras_available:
            prob = float(self.model.predict(point, verbose=0)[0][0])
            pred = 1 if prob >= 0.5 else 0
            conf = prob if pred == 1 else (1 - prob)
        else:
            probs = self.model.predict_proba(point)[0]
            pred = int(self.model.predict(point)[0])
            conf = float(probs[pred])
        return pred, conf

    def save_model(self, file_path: str):
        if self.engine_type == "TensorFlow/Keras" and keras_available:
            self.model.save(file_path + ".h5")
        else:
            joblib.dump(self.model, file_path + ".pkl")
            
    @staticmethod
    def is_keras_supported():
        return keras_available

📝 Подробный разбор кода model_engine.py

  • Блок try-except для TensorFlow: Если библиотека `tensorflow` не установлена на локальном компьютере, программа не падает с критической ошибкой, а переводит флаг `keras_available` в состояние `False` и переключается на встроенный в Python классификатор Scikit-learn MLP. Это гарантирует надежность работы приложения в учебных лабораториях.
  • Динамическое создание слоев: Конструктор Keras инициализирует модель `Sequential` (последовательность слоев), считывая конфигурацию количества скрытых слоев и нейронов прямо из виджетов Streamlit. Последний слой имеет 1 нейрон и функцию активации `sigmoid`, так как мы решаем задачу бинарной классификации (выдаем вероятность принадлежности точки к Зеленому классу от 0 до 1).
  • Пошаговое обучение по эпохам: Мы не вызываем метод `fit()` на 100 эпох сразу, иначе мы бы не смогли выводить живые графики во время расчета. С помощью метода `partial_fit()` в Scikit-learn или запуска обучения Keras на 1 эпоху (`epochs=1`) внутри внешнего цикла `for` мы получаем возможность опрашивать модель после каждого шага оптимизации, считывать текущую ошибку (`loss`) и точность (`accuracy`) и отправлять их в веб-интерфейс.
  • Магия Mesh Grid (Координатная сетка): Для визуализации границы решения мы создаем матрицу плоскости. Метод `np.meshgrid` строит сетку из точек с шагом разрешения (например, $50 \times 50 = 2500$ точек). Мы подаем координаты этих 2500 точек в обученную нейросеть с помощью `predict_proba`. Модель выдает массив вероятностей `Z`, который превращается с помощью контурного графика `contourf` в цветную тепловую карту разделения пространства.

🌐 Шаг 4 — Разработка графического веб-интерфейса (app.py)

Этот файл является центральным диспетчером. Он импортирует два созданных ранее модуля, верстает боковую панель управления со всеми ползунками, настраивает вкладки дашборда и запускает анимацию обучения нейросети. Создайте в PyCharm Python-файл app.py и добавьте следующий код:

app.py python 
import streamlit as st
import pandas as pd
import numpy as np
import matplotlib.pyplot as plt
import time
import os
from sklearn.metrics import ConfusionMatrixDisplay
from data_manager import DataManager
from model_engine import ModelEngine

# 1. Инициализация конфигурации страницы
st.set_page_config(page_title="Modular AI Playground", page_icon="🧠", layout="wide")

# 2. Инициализация состояния сессии для хранения результатов при перезагрузке страницы
if "model_engine" not in st.session_state:
    st.session_state.model_engine = None
    st.session_state.metrics = None

st.title("🧠 Модульный AI-симулятор: Проектирование и обучение нейросетей")

# --- БОКОВАЯ ПАНЕЛЬ НАСТРОЕК (Sidebar) ---
st.sidebar.title("🛠️ Панель конфигурации")
engine_option = st.sidebar.selectbox("Вычислительное ядро:", ["Scikit-learn (MLP)", "TensorFlow/Keras"] if ModelEngine.is_keras_supported() else ["Scikit-learn (MLP)"])
engine_type = "TensorFlow/Keras" if "TensorFlow" in engine_option else "Scikit-learn (MLP)"

dataset_type = st.sidebar.selectbox("Набор данных:", ["Полумесяцы (make_moons)", "Круги (make_circles)", "Блоки (make_blobs)"])
n_samples = st.sidebar.slider("Число точек:", 100, 1000, 400, step=50)
noise_val = st.sidebar.slider("Уровень шума:", 0.05, 0.45, 0.15, step=0.05)
test_split = st.sidebar.slider("Размер теста (доля):", 0.1, 0.4, 0.2, step=0.05)

neurons_l1 = st.sidebar.slider("Нейронов в 1 слое:", 4, 32, 16, step=4)
neurons_l2 = st.sidebar.slider("Нейронов во 2 слое:", 0, 16, 8, step=2)
activation_fn = st.sidebar.selectbox("Активация нейронов:", ["relu", "tanh", "logistic", "identity"])
epochs_val = st.sidebar.slider("Количество эпох:", 5, 100, 30, step=5)
lr_val = st.sidebar.selectbox("Шаг обучения (LR):", [0.001, 0.01, 0.1, 0.5], index=1)

hidden_layers = (neurons_l1, neurons_l2) if neurons_l2 > 0 else (neurons_l1,)

# 3. Вызов методов модуля DataManager для подготовки выборки
X, y = DataManager.generate_dataset(dataset_type, n_samples, noise_val)
X_train, X_test, y_train, y_test = DataManager.prepare_data(X, y, test_split)

tab1, tab2, tab3 = st.tabs(["📊 Модуль 1: Данные", "📈 Модуль 2: Обучение", "🏆 Модуль 3: Тестирование"])

# --- ВКЛАДКА 1: Модуль данных (анализ таблиц Pandas) ---
with tab1:
    st.header("Модуль данных (data_manager.py)")
    df = pd.DataFrame(X, columns=["X1", "X2"])
    df["target"] = y
    st.dataframe(df, use_container_width=True)
    
    class_summary = DataManager.get_metrics_summary(df)
    st.write("Распределение классов:")
    for cls, count in class_summary.items():
        st.info(f"Класс {cls}: {count} точек")

# --- ВКЛАДКА 2: Живая визуализация обучения (Matplotlib + Streamlit) ---
with tab2:
    st.header("Живая визуализация обучения")
    if st.button("🚀 Начать обучение"):
        engine = ModelEngine(engine_type, hidden_layers, activation_fn, lr_val)
        engine.initialize_model(input_dim=2)
        
        # Резервирование пустых контейнеров в веб-интерфейсе для предотвращения дублирования графиков
        plot_loss = st.empty()
        plot_acc = st.empty()
        progress = st.progress(0)
        
        history_loss, history_acc = [], []
        classes = np.unique(y_train)
        
        # Цикл пошагового обновления графиков
        for epoch in range(epochs_val):
            loss, acc = engine.train_epoch(X_train, y_train, classes=classes)
            history_loss.append(loss)
            history_acc.append(acc)
            
            # Отрисовка графика ошибки Loss
            fig_l, ax_l = plt.subplots(figsize=(8, 2.5))
            ax_l.plot(history_loss, color="#e74c3c", label="Loss", marker="o", markersize=2)
            plot_loss.pyplot(fig_l)
            plt.close(fig_l)
            
            # Отрисовка графика точности Accuracy
            fig_a, ax_a = plt.subplots(figsize=(8, 2.5))
            ax_a.plot(history_acc, color="#2ecc71", label="Accuracy", marker="x", markersize=2)
            plot_acc.pyplot(fig_a)
            plt.close(fig_a)
            
            progress.progress(int(((epoch+1)/epochs_val)*100))
            time.sleep(0.05)
            
        st.session_state.model_engine = engine
        st.session_state.metrics = engine.evaluate(X_test, y_test)
        st.success("Обучение завершено!")
        st.balloons() # Эффект праздничных шаров

# --- ВКЛАДКА 3: Оценка результатов, тепловая карта и ИИ-Тестер ---
with tab3:
    st.header("Результаты оценки и Тестер")
    if st.session_state.model_engine is not None:
        engine = st.session_state.model_engine
        metrics = st.session_state.metrics
        
        st.write(f"Accuracy на тесте: **{metrics['accuracy']*100:.2f}%**")
        
        col_res1, col_res2 = st.columns(2)
        
        with col_res1:
            xx, yy, Z = engine.get_decision_boundary(X, resolution=50)
            fig_map, ax_map = plt.subplots(figsize=(6, 4.5))
            ax_map.contourf(xx, yy, Z, alpha=0.8, cmap="RdYlGn")
            ax_map.scatter(X[:,0], X[:,1], c=y, edgecolors="k", cmap="RdYlGn_r")
            st.pyplot(fig_map)
            plt.close(fig_map)
            
        with col_res2:
            st.subheader("📊 Матрица ошибок (Confusion Matrix)")
            fig_cm, ax_cm = plt.subplots(figsize=(5, 4.5))
            disp = ConfusionMatrixDisplay(confusion_matrix=metrics['confusion_matrix'], display_labels=["Класс 0", "Класс 1"])
            disp.plot(ax=ax_cm, cmap="Blues", colorbar=False)
            st.pyplot(fig_cm)
            plt.close(fig_cm)
            
        st.markdown("---")
        
        # Интерактивный ИИ-Тестер для проверки координат пользователем
        st.subheader("🔮 Проверь собственную точку")
        p_x1 = st.number_input("Координата X1:", value=0.0)
        p_x2 = st.number_input("Координата X2:", value=0.0)
        
        if st.button("🔍 Узнать класс точки"):
            test_point = np.array([[p_x1, p_x2]])
            pred_class, conf_val = engine.predict_point(test_point)
            
            if pred_class == 1:
                st.success(f"🟢 Точка относится к Классу 1. Уверенность: {conf_val*100:.2f}%")
            else:
                st.error(f"🔴 Точка относится к Классу 0. Уверенность: {conf_val*100:.2f}%")

📝 Подробный разбор кода app.py

  • Сессии Streamlit (st.session_state): Streamlit устроен так, что при любом взаимодействии пользователя со страницей (клик по кнопке, сдвиг ползунка) весь скрипт `app.py` перезапускается сверху вниз. Чтобы при вводе координат на 3-й вкладке обученная модель не удалялась из памяти, мы сохраняем объект математического ядра в сессионную память: `st.session_state.model_engine = engine`. Она сохраняет состояние данных при перезагрузках страницы.
  • Резервирование контейнеров (st.empty): Если вызывать отрисовку графиков `st.pyplot()` напрямую в цикле обучения, Streamlit будет каждую эпоху рисовать новый график снизу предыдущего. На экране образуется длинная лента из картинок. Метод `st.empty()` создает пустую ячейку, содержимое которой мы заменяем новым графиком каждую эпоху с помощью `.pyplot()`, создавая плавный эффект изменения кривой.
  • Матрица ошибок (Confusion Matrix Display): В модуле тестирования используется встроенный класс визуализации `ConfusionMatrixDisplay`. Он берет предсказания на тестовой выборке и сравнивает их с правильными ответами, формируя таблицу $2 \times 2$. Студенты видят точное количество True Positive, True Negative, False Positive и False Negative ответов сети на тесте.

🚀 Шаг 5 — Запуск разработанного веб-приложения

Запуск Streamlit отличается от запуска обычных Python-файлов. Веб-сервер запускается специальной командой через командную строку.

Пошаговый запуск в консоли PyCharm:

  1. В нижней части экрана PyCharm откройте вкладку встроенного терминала: нажмите кнопку Terminal (или используйте горячие клавиши Alt + F12). Обратите внимание, что терминал PyCharm автоматически активирует виртуальное окружение (venv).
  2. Введите в открывшейся строке терминала команду запуска сервера:
Терминал PyCharm bash 
streamlit run app.py
  1. Нажмите клавишу Enter. В консоли отобразятся системные логи запуска, а в браузере автоматически откроется новая вкладка по локальному веб-адресу: http://localhost:8501.
⚠️
Полезный совет: Вам не нужно перезапускать сервер в терминале каждый раз, когда вы вносите правки в файлы `data_manager.py`, `model_engine.py` или `app.py`. Просто сохраните изменения в PyCharm, обновите страницу в браузере, и Streamlit мгновенно подгрузит обновленный код.
